# MS-365-MFA-config
Repo showing my knowledge of MFA within MS365

<h2> Enabling MFA through Entra ID (formerly Azure Active Directory) </h3>

<div align="center"> Within "Microsoft 365 admin center", select the drop down arrow of the "Users" category.
Next, select the "Active users" sub-category. Under the "Active Users" title, select "Multi-factor authentication".</div>


<p align="center">
    <img src="https://github.com/zay65/MS-365-MFA-config/blob/37e825a488ce683671511e0e82213499edb008de/MS%20365%20MFA%201.png" alt="Sample Image"/>
  </p>

<div align="center"> Select the name(s) of the user(s) you wish to enable MFA for, select "Enable MFA" (above "Name"), & then when prompted to "Enable multifactor authentication", select the "Enable" button.</div>


<p align="center">
    <img src="https://github.com/zay65/MS-365-MFA-config/blob/392de71332bd6c0a43fe19d3bb0142beec04618a/MS%20365%20MFA%202.png" alt="Sample Image"/>
  </p>

<div align="center"> Multifactor authentication has been enabled.</div>
